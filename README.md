# Qt5-CMake Starter
A basic scaffold for setting up a Qt5 project with CMake.

## Install
```sh
$  git clone git@github.com:codymj/qt5-cmake-starter.git
$  cmake CMakeList.txt
$  make
```

## Usage
```sh
$  ./qt5-cmake-starter
```
A GUI should appear, allowing you to click the "Hello, World!" button to end the
program.
